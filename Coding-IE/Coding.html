KOJO

Kojo è una piattaforma open source che ti permette di disegnare grafiche.
È stata utilizzata ad inizio anni e questi sono alcuni dei lavori svolti:

SHAPE-BLOCK

La tecnica SHAPE-BLOCK consiste nel creare prima una funzione "Shape", dove si va a definire una figura, e poi una funzione "Block" che va a personalizzare Shape, ruotandola e colorandola. Infine ci sarà lo script principale che andrà a ruotare block x volte, formando una specie di fiore

TEXTURE

Una volta ottenuto l'oggetto che vogliamo disegnare, possiamo ripeterlo creando una griglia che occupa tutto lo schermo. È possibile fare ciò usando i CanvasBounds, ovvero i confini dello schermo, e poi basare tutte le misure su di essi.

HTML

HTML è un linguaggio usato da utto il mondo per fare pagine web, e, accompagnato da CSS e JavaScript, può creare pagine web stilizzate e interrative.
Questa stessa pagina è stata fatta con HTML
Questi sono alcuni lavori fatti da noi:

Pagine Web

La prima cosa che abbiamo imparato a fare sono pagine web semplici; hanno titoli, paragrafi, immagini e link.

Stilizzazione con CSS

Noi abbiamo utilizzato CSS proprio per personalizzare i nostri siti HTML. Dopo aver conosciuto le varie variabili del linguaggio CSS abbiamo modificato la struttura del file HTML collegandolo ad un file CSS per dargli una formattazione diversa per ogni alunno.

BootStrap

Infine abbiamo usato BootStrap, che è sostanzialmente un codice CSS già personalizzato da una community di designer, in modo tale che tu puoi inserire solo i contenuti e farlo sembrare professionale
